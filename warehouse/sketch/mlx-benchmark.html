Key Insights

Thinking Waste

0%
3B Coder is the only model with zero thinking overhead. Every token is output.

Qwen3.5 Thinking OFF

Configurable
Unlike LFM8B, Qwen3.5 thinking can be disabled via chat template config. Works correctly.

LFM8B Ticket

67-100%
67-100% token waste on every task. Thinking is hard-baked at architecture level, cannot be disabled.

RAM Efficiency

1.5 GB
LFM8B wins on RAM (1.5 GB for 8B MoE params). But useless if 90%+ of output is thinking.

Heaviest Model

4.6 GB
Qwen3-8B uses the most RAM (4.6 GB, 28% of 16GB). Pure attention arch, no thinking waste. Predictable ~17 tok/s throughput across all tasks.

Multi-step

Qwen wins
Qwen3.5 handles multi-step tool planning (2 tools in one turn). LFM8B fails at this.

Format drift

LFM wins
LFM8B maintains format across turns. Qwen3.5 drifts on follow-up calls.

Verdict

Three models, three roles. Here's when to use each.

Qwen2.5-Coder-3B

Daily Driver · King

Use for everything utility: codegen, HTML, summarization, filtering, CSS, regex, bash. Fastest, cleanest, lowest RAM, zero overhead. Not designed for agentic tasks.

Qwen3.5-4B-OptiQ

Agentic Complement

Use when you need agentic capability: multi-step tool plans, JSON output, structured formats. Thinking OFF. 2x slower, heavier RAM. Stronger output quality on prose/summary.

LFM2.5-8B-A1B

Specialist · Benchwarmer

Use only when you need chain-of-thought reasoning and can spare 4096 tokens. Best RAM efficiency for its size. Useless for utility tasks under 1024 budget. Keep for specialist use.

Qwen3-8B-4bit

New Challenger · Heavyweight

Pure attention architecture, no thinking waste, predictable throughput. Complete output guarantee at 1024 budget — never truncated. But eats 4.6 GB RAM and runs 2-3x slower than 3B Coder. Best for tasks where completeness matters more than speed.

This Is the Way

3B Coder stays king. Qwen3.5-4B enters as agentic complement. LFM8B stays as specialist depth. Qwen3-8B joins as heavyweight challenger — complete output at any budget but 4.6 GB RAM is steep. Four models, four lanes.
