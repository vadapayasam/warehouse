Key Insights

Thinking Waste

0%
3B Coder and Qwen3-8B are pure attention — zero thinking overhead. Qwen3.5-4B configurable OFF. LFM8B: 67-100% hard-baked waste.

Qwen3.5 Thinking OFF

Configurable
Unlike LFM8B, Qwen3.5 thinking can be disabled via chat template config. Works correctly.

LFM8B Ticket

67-100%
67-100% token waste on every task. Thinking is hard-baked at architecture level, cannot be disabled.

RAM Efficiency

1.5 GB
LFM8B wins on RAM (1.5 GB for 8B MoE params). But useless if 90%+ of output is thinking.

Heavy Synthesis

8B / 4.7 GB
Qwen3-8B: same speed as Qwen3.5-4B (~19 tok/s) but 2.4x more params for heavier lifting. Pure attention, no thinking waste. Dedicated synthesis crons.

Multi-step

Qwen wins
Qwen3.5 handles multi-step tool planning (2 tools in one turn). Qwen3-8B ~67% consistent. LFM8B fails.

Format drift

Qwen3-8B wins
Qwen3-8B maintains format across turns with native tool result routing. Qwen3.5 drifts on follow-up calls. LFM8B holds format but with thinking waste.

Native tool calling

Qwen3-8B
Only model with native tool_calls API in RapidMLX. Pure function call format — no thinking blocks, no prose, no format drift on continuation. 2.8s per call.

Verdict

Three models, three roles. Here's when to use each.

Qwen2.5-Coder-3B

Daily Driver · King

Use for everything utility: codegen, HTML, summarization, filtering, CSS, regex, bash. Fastest, cleanest, lowest RAM, zero overhead. Not designed for agentic tasks.

Qwen3.5-4B-OptiQ

Agentic Complement

Use when you need agentic capability: multi-step tool plans, JSON output, structured formats. Thinking OFF. 2x slower, heavier RAM. Stronger output quality on prose/summary.

Qwen3-8B-4bit

Synthesis Workhorse + Tool Calling

Use for heavy synthesis: long-form summaries, batch research processing, any task needing 8B params with zero thinking waste. Same speed as Qwen3.5-4B but 2.4x more params. Native tool_calls API — best multi-turn adherence, cleanest tool format. Heaviest RAM at 4.7 GB — keep for dedicated synthesis crons.

LFM2.5-8B-A1B

Specialist · Benchwarmer

Use only when you need chain-of-thought reasoning and can spare 4096 tokens. Best RAM efficiency for its size. Useless for utility tasks under 1024 budget. Keep for specialist use.

This Is the Way

3B Coder stays king. Qwen3.5-4B stays as agentic complement. Qwen3-8B enters as synthesis workhorse — same speed as Qwen3.5-4B but 2.4x more params for heavier lifting. LFM8B stays as specialist depth. The hierarchy grows, no one replaced.
