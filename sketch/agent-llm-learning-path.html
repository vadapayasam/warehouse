The Physics Layer

LLM agents exhibit phase transitions. Below a certain model scale, adding compute to reasoning does nothing. Above the threshold, chain-of-thought suddenly kicks in. Understanding this helps you pick model sizes for automation tasks and avoid over-engineering solutions that work fine with smaller models.
