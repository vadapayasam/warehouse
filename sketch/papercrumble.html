V1 How It Works

  • CrumpleGeometry — single Double progress (0→1) drives all visual properties: size, wiggle, shadow, float offset, rotation, content opacity.
  • 4 Path Generators — one for each stage band:
    • crumpledBallPath: polygonal blob with wiggled radius
    • roughRectanglePath: jagged straight-line segments
    • warpedRectanglePath: rounded rect with bezier wave edges
    • subtleWavePath: nearly perfect rounded rect with tiny top/bottom wave
  • CreaseOverlay — Canvas-drawn diagonal creases + fine wrinkles, faded by progress. Clipped to the paper shape.
  • PaperCardContent — LeaveHack calendar UI (header, mini grid, legend, CTA) with opacity keyed to progress.
  • InteractiveCrumpleCard — tap toggles between progress 0 and 1 with spring animation.
