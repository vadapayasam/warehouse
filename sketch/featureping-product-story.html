What FeaturePing does

Release note in. In-product explainer out.

1Paste the release note

Raw engineering or PM notes — whatever you have.

2AI drafts the user copy

Page + element, e.g. button[data-action='bulk-export']. Verified before publish.

Where it stands

Working prototype. Built, tested, honest about the gaps.

  • Framework-agnostic widget — one script tag, any stack
  • “Show me” highlight, including cross-page navigation
  • Admin composer with preview & target verification
  • Schema-validated update contract + exports
  • API prototype, unit tests across the whole flow

Deliberately not yet
