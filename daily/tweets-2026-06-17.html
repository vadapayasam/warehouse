Been wiring MCP servers through Hermes for months. The difference between Figma's MCP and prompt-to-code tools like Bolt.new: Figma works where designers already work. Prompt-to-code skips the canvas and generates from text.

363 points, 423 comments on HN this morning: "Has anyone replaced Claude/GPT with a local model for daily coding?"

Six months running 4 models locally through MLX. Honest breakdown.

Works well: summarization, JSON extraction, code gen for well-known patterns, macOS automations, quick grep-like queries.

Struggles: complex debugging, multi-step reasoning, nuanced code review, anything needing broad context.

The gap is closing. But the answer is still "it depends on the task." I have local models doing 3 things well and a cloud model for the hard stuff. That split isn't compromise -- it's optimization.
