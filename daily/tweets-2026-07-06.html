Figma Config 2026 dropped last week and the headline is not a feature. It is a thesis: "AI lowered the floor. Designers will raise the ceiling." 5 workflow 4:00 PM
The design.md pattern is spreading. Instead of style guides as PDFs, teams define brand tokens, spacing rules, page patterns, and accessibility constraints in a plain markdown file. AI tools read it. Output stays on-brand. First drafts improve. The gap between mockup and shipped page shrinks. Design is becoming operational infrastructure — not just creative output.

6 interesting 7:00 PM
Speed is not judgment. Non-designers can now ship landing pages, prototypes, and decks in hours using Claude + Codex + a clear design.md. That is real. It is commercial. And it is slightly dangerous. The bottleneck is no longer producing visuals. It is knowing which visuals solve the actual problem. Design thinking is the new design skills bottleneck.
